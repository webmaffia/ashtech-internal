STEP 2: ACTIVATE IN WORDPRESS
------------------------------

1. Login to WordPress Admin
2. Go to Plugins
3. Find "Ashtech Gutenberg Blocks"
4. Click Activate
5. If already active, Deactivate then Activate again

1. Pages → Add New
2. On right sidebar → Template dropdown
3. Select "Ashtech Full Width (No Wrappers)"
4. Add your Ashtech blocks
5. Publish
