## ğŸš¨ Common Issues

### Issue: Blocks not showing?
**Fix:** Rebuild plugin
```bash
npm run build
```

### Issue: Styles not applied?
**Fix:** 
1. Make sure template is "Ashtech Blank"
2. Hard refresh: Ctrl+Shift+R
3. Clear cache

### Issue: Images not loading?
**Fix:**
1. Check file size (< 2MB)
2. Use supported formats (JPG, PNG, SVG)
3. Upload via Inspector Controls

### Issue: Header overlaps banner?
**Fix:** This is normal! Header is fixed.
- Banner should be full height
- Header will be transparent over it
